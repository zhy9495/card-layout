# 风格包 · 包豪斯几何 (bauhaus)

> 母题:**Bauhaus 包豪斯**——三原色 + 基本几何形 + 无衬线大字 + 不对称构成。
> 灵魂:理性、几何、大胆;靠形与色的构成,不靠装饰。

## 1. 配色
`--paper:#EFE9DC` · `--ink:#1A1A1A` · 三原色 `--red:#E2412B` `--blue:#2B57C4` `--yellow:#F2C200`。
铁律:只用红/蓝/黄三原色 + 黑,纯色块、硬边、不渐变。一张卡 1-3 个几何形点睛。

## 2. 字体
标题 思源黑体 Noto Sans SC **900**(几何无衬线);正文 思源黑 / Inter。层级靠字号 + 一个原色。
引入:`https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@400;900&family=Inter:wght@400`

## 3. 招牌动作
① **三原色几何形**(圆/方/三角)作构成元素,不对称摆放;② 黑色横杠/线做节奏;③ 网格页脚(`border-top:3px`);④ 大字 flush-left,关键词标红。

## 4. 氛围
无纹理、无投影、无圆角。秩序与构成即氛围。

## 5. 页面集(模块)
`bauhaus.html`(封面)+ `page-body/concept/data/compare/steps/quote/list/back.html` + `page-img-ab/lr/duo/grid.html`,各带 `sample-*.png`。
- 数据=超大字 + 圆;对比=中缝竖线双栏 + 方/圆;步骤·清单=彩色方/圆编号;正文档=常规粗细 + 黄高亮 + 小几何。
- **图文**:方角(radius 0)图片,可配色块边;布局见全局图文规则。

## 6. do / don't
✅ 只三原色+黑、几何点睛、大字左对齐、硬边。
❌ 不加第四色、不加纹理/渐变/圆角(图文除外可直角)、几何别撒满。

## 7. 适合
设计/思维/创意/知识/方法论。不适合温柔治愈、文艺抒情。
