# 风格包 · 瑞士国际主义网格 (swiss-grid)

> 母题:**Swiss / 国际主义平面设计**(Müller-Brockmann、Akzidenz/Helvetica 传统)。
> 灵魂一句话:**理性的网格 + 无衬线大字 + 红黑白**——靠秩序和留白显高级,不靠装饰。
> 经典平面设计第一课。和 warm-editorial(暖衬线)、folder-doodle(俏皮)完全不同:**冷静、专业、几何**。

---

## 1. 配色(红黑白,经典三色)

| token | 值 | 用途 |
|---|---|---|
| `--paper` | `#F2F0EA` | 暖白底(可换纯白 #FFFFFF) |
| `--ink` | `#141414` | 标题/正文/分隔线 |
| `--red` | `#E2231A` | **唯一强调=国际主义红**:几何方块 + 1 个关键词 + 标签 |

**铁律**:只有红黑白。红用得**狠而少**(一个方块 + 一个词),其余全黑白。绝不加第二色。

---

## 2. 字体(纯无衬线,这套不用衬线)

```
标题:思源黑体 Noto Sans SC 900(Helvetica 的中文等价)/ 英文 Inter 700
正文/标签:思源黑体 400-500 / 英文 Inter
```
引入:`https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@400;500;700;900&family=Inter:wght@400;600;700`

字号(@1080×1440):
- 主标题:`126–140px` / line-height `1.05` / 900 / `letter-spacing:-.01em` / **flush-left(左对齐顶左)**
- 副标:`32–34px` / 400
- 顶/底标签:`25px` / 500 / `letter-spacing:.1em`

---

## 3. 招牌动作(必须有)

1. **严格网格** —— 顶部一条粗分隔线(`border-bottom:3px`)带标签行(左中文标签 / 右红色英文词);底部一条线带页脚(@handle / 页码)。
2. **几何红方块** —— 右上角一个纯红正方形(~168px),平衡左重的大标题(Swiss 的不对称张力)。
3. **无衬线大字 flush-left** —— 标题顶天立地、左对齐、字间收紧;关键词 1 个标红。
4. **大留白** —— 内容压上半/左,右下大片空。

---

## 4. 氛围层

无纹理、无投影、无装饰。**秩序本身就是氛围**。可选:一条粗黑短 rule(`width:300px;heightः10px`)做节奏停顿。

---

## 5. 版式骨架(1080×1440 · flush-left · 网格)

```
─────────────────────────────  ← 顶粗线
[中文标签 左 ·············· 红色英文词 右]
                            [红方块 ■ 右上]
背单词
靠[间隔(红)]              ← 大字 flush-left,垂直占上半
不靠遍数
▬▬▬▬                       ← 粗黑短 rule
[副标 2 行]

[@handle 左 ········ 01/06 右]
─────────────────────────────  ← 底线
```
边距 `90px`。一切左对齐,右侧留白由红方块点睛。

---

## 6. 实现要点

- 顶/底标签行用 `display:flex; justify-content:space-between` + `border-bottom/top`。
- 红方块 `position:absolute` 右上;页码用 Inter(等宽感)。
- 渲染走 §8 管线;本套字体走 Google Fonts,`--virtual-time-budget=12000`。

---

## 7. do / don't

- ✅ 只红黑白、大字顶格左对齐、严格网格线、红用得少而狠、大留白。
- ❌ 不加第二色、不加纹理/投影/圆角、不居中堆叠、不用衬线(这套就是无衬线的)、标题别缩小到不够冲。

---

## 8. 适合 / 不适合

适合:**知识 / 方法 / 清单 / 观点 / 效率职场** —— 要"专业可信、清晰有力"的内容。
不适合:温柔治愈、文艺抒情(找 warm-editorial / vintage-collage)、热闹俏皮(找 folder-doodle)。
