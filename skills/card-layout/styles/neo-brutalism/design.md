# 风格包 · 新粗野主义 (neo-brutalism)

> 母题:**Neo-Brutalism**——粗黑边框 + 硬质偏移投影 + 平涂撞色块。
> 灵魂:冲、直接、no-nonsense;原始有力。

## 1. 配色
`--paper:#F4EFDF`(圆点底) · `--ink:#111` · 亮色 `--yellow:#FFD23F` `--pink:#FF5CA8` `--cyan:#4ECDC4`。
平涂亮色块,黑边黑影,够亮够冲。

## 2. 字体
标题 思源黑 **900**;正文 思源黑 / Inter。

## 3. 招牌动作
① **粗黑边框**(`border:6-7px solid #111`)+ **硬投影**(`box-shadow:14px 14px 0 #111`);② 平涂撞色块/贴纸(可轻微旋转);③ 圆点底纹(`radial-gradient`);④ 大字 900。

## 4. 氛围
圆点底 + 硬投影。不要柔和、不要渐变、不要细腻。

## 5. 页面集(模块)
`neo.html`(封面)+ `page-body/concept/data/compare/steps/quote/list/back.html` + `page-img-*.html`。
- 内容装进带边框+投影的盒子;对比=两个撞色盒;步骤=带框编号条;正文档=白盒内常规粗细 + 黄高亮。
- **图文**:图片本身 `border:6px solid #111 + box-shadow` 偏移投影(直角),这就是本套图框签名。

## 6. do / don't
✅ 粗边 + 硬影 + 撞色 + 圆点底。
❌ 不圆角(柔)、不渐变、不柔和投影、撞色别太多种。

## 7. 适合
清醒/观点/成长/干货/态度。不适合温柔文艺、安静克制。
