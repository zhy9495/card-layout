# 风格包 · 打字机便签 (typewriter-memo)

> 母题:**Typewriter / 索引卡备忘**——横线纸 + 等宽字 + 盖戳。
> 灵魂:低调、亲密、手作;像写给自己的一张便签。

## 1. 配色
`--paper:#F1ECDD`(纸纹) · `--ink:#2B2824` · 盖戳 `--stamp:#B23A2E` · `--muted:#9C9484`。

## 2. 字体
JetBrains Mono(拉丁/数字等宽)+ 思源黑体(中文),正文标题同款,营造打字机感。
引入:`JetBrains+Mono:wght@400;700 & Noto+Sans+SC:wght@400;500`

## 3. 招牌动作
① **横线纸**(每行 `border-bottom`)+ 纸纹噪点;② **MEMO 抬头 + 编号**(虚线分隔);③ **红色盖戳**(旋转矩形,如「已收到」);④ 等宽字 + `[ ]` `☐` 符号。

## 4. 氛围
纸纹 + 横线 + 盖戳 = 手作备忘感。

## 5. 页面集(模块)
`memo.html`(封面)+ `page-body/concept/data/compare/steps/quote/list/back.html` + `page-img-*.html`。
- 概念=`[ 词 ]`;清单=`☐` 勾选;数据=等宽大数字;金句=盖戳;正文档=横线上逐行写。
- **图文**:图片 `border:12px solid #fff` + 柔投影 = **照片冲印/拍立得**质感(贴在便签上)。

## 6. do / don't
✅ 横线 + 等宽 + 盖戳 + 纸纹 + 方括号符号。
❌ 不用花哨彩色、不用粗黑大字、盖戳别超过 1 个。

## 7. 适合
自我对话/情绪/治愈/日记/成长。不适合需要冲击力或专业数据感。
