# 风格包 · 暖纸编辑感 · 高对比衬线 (warm-editorial)

> 母题:**Didone 衬线报头**(Vogue / Harper's 那种粗细对比极强的时尚刊大标题)× **当代个人品牌连发**的留白美学。
> 灵魂一句话:**像一本安静的杂志**——只用一个锈红、敢留大片白、衬线大标题挑大梁,手绘箭头加一点人味。
> 拆解来源:Richard Sanchez「Step by Step Career Pivot」九宫格(2026-06-24 用户提供)。

---

## 1. 配色(克制是第一原则)

| token | 值 | 用途 |
|---|---|---|
| `--paper` | `#EFE9DC` | 暖米/骨白,永远是底 |
| `--ink` | `#1F1B16` | 标题 + 正文(暖近黑,不要纯黑 #000) |
| `--accent` | `#B85C44` | 锈陶红,**只用在**:品牌名、大数字、小下划线、图标、页脚、头像圈 |
| `--line` | `#1A1A1A` | 手绘箭头/手绘线 |

**铁律:全卡只有一个强调色。** 锈红出现面积要小而精(数字、点缀),靠**大留白 + 黑米对比**撑画面,不靠多色。**禁止**再加第二个强调色。`--mute` 走 0%(原色),这套本来就低饱和,别再柔化。

## 2. 字体(高级感的一多半)

```
全卡只用一种字:'KingHwaOldSong'(京华老宋体)—— ★ 老铅字对比/文气,免费商用
  标题 / 正文 / 标签 / 页脚 / 大数字 全部用它,英文 fallback 'Playfair Display'
层级只靠「字号 + 锈红色」拉开,不靠换字、不靠字重(京华仅一个字重)
```
> 之前是「衬线标题 + 黑体正文」两种字;现已按用户要求**统一成一种字**(2026-06-24)。要回到双字混排再说。

引入(京华老宋体走 ZeoSeven CDN + 其余 Google Fonts,两条 link):
```html
<link href="https://fontsapi.zeoseven.com/309/main/result.css" rel="stylesheet">
<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;700;900&family=Noto+Serif+SC:wght@400;700;900&family=Noto+Sans+SC:wght@400;500;700&family=Inter:wght@400;600&display=swap" rel="stylesheet">
```
> 京华老宋体只有一个字重 → 标题 `font-weight:400`(别设 900,无效)。思源宋 900 作为 CDN 挂掉时的 fallback:`font-family:'Playfair Display','KingHwaOldSong','Noto Serif SC',serif`。

字号阶梯(@1080×1440,**字要大、对比要狠**):
- 封面主标题:`128–135px` / line-height `1.08`
- 内页大数字:`150–160px` / 锈红
- 内页标题:`80–88px` / line-height `1.12`
- 领句:`34–36px`(靠字号 + 留白和子条目拉开,不加粗)
- 子条目正文:`30–32px` / line-height `1.7`
- 品牌名 + 页脚:`27px` / 锈红 / letter-spacing `.05em`

> ⚠️ 中文衬线标题用 **京华老宋体**:横细竖粗、有老铅字文气,已很接近 Didone 编辑感(比思源宋好一档)。**英文标题(英语学习内容)直接 Playfair = 和原图零差别**。
> ⚠️ 衬线左对齐时左缘内缩,统一加 `margin-left:-0.04em` 对齐到上下的黑体那条直线。

## 3. 招牌动作(必须有,缺了就不是这套)

1. **超大锈红数字编号** —— 步骤/清单卡正文区左上方,Playfair 900,压在标题上方一点。
2. **手绘波浪箭头** —— 黑色、随意、像马克笔。SVG 路径见 §6。可**跨卡流动**:这张右缘箭头指出 → 下一张承接(连发时增强"翻页"感)。每张最多 1 根,别撒多。
3. **只用一个强调色 + 大留白** —— 内容压上半/左上,下半留大片空白。这是"透气=高级"的根。

---

## 4. 氛围层(反常识:几乎不加纹理)

这套**不靠**纸纹噪点做旧。质感来自**留白 + 克制 + 那根手绘箭头**。
- ❌ 不要纸纹、不要噪点、不要做旧滤镜 —— 会脏,破坏干净的编辑感。
- ✅ 可选:极细 hairline 分割线(`1px` ink 20% 透明)、页脚一道 `40px` 锈红短下划线。仅此。

---

## 5. 版式骨架(1080×1440 · 全部左对齐 · 三段撑满整张)

> 核心:**别把内容全顶到上面**。用「品牌顶 / 主内容垂直居中 / 页脚底」三段占满整张高度,内容才不显局促。
> 实现:`body{position:relative}`;品牌 `absolute top:96 left:90`;主内容 `absolute top:50% translateY(-52%)`;底栏 `absolute bottom:96`。左右边距 `90px`。

**A. 封面**
```
[品牌名 · 锈红 · 左上(absolute)]

           [大标题 · 2 行 · 132px]      ← 垂直居中
           [副标 · 34px · 紧贴标题下]

[页脚锈红小字 + 短下划线 · 左下(absolute)]
```

**B. 步骤 / 清单内页(对应 card-layout B 家族)**
```
[品牌名 · 锈红 · 左上(absolute)]

           [大数字 · 锈红 · 158px]      ← 主块垂直居中
           [标题 · 84px · 1 行]
           [领句 · 36px]
           [· 子条目 ×3]

[页脚锈红小字 ········· 手绘箭头]  ← 底栏一行:左文右箭头,同一水平线(平行)
```
> 底栏用 `display:flex; justify-content:space-between; align-items:center`,页脚与箭头**务必同一基线**(用户明确要求)。

**C. 概念 / 金句内页**
- 概念:去掉数字,衬线标题 + 一段领句 + 短说明,留白更大。
- 金句:衬线大字居中偏左、`72–84px`,锈红一根短下划线在下方;无数字无箭头。

**封底**:大衬线标题(如"记得收藏")+ 一行副标 + 锈红头像圈 + 品牌名 handle(呼应原图末卡)。

## 6. 实现素材

**手绘波浪箭头(inline SVG,锈黑可调):**
```html
<svg class="hand-arrow" viewBox="0 0 180 60" fill="none" xmlns="http://www.w3.org/2000/svg">
  <path d="M6 34 C 40 12, 70 56, 104 30 S 150 20, 168 32" stroke="var(--line)"
        stroke-width="3.2" stroke-linecap="round"/>
  <path d="M150 18 L170 32 L150 46" stroke="var(--line)" stroke-width="3.2"
        stroke-linecap="round" stroke-linejoin="round" fill="none"/>
</svg>
```
（弯度/方向可镜像:`transform: scaleX(-1)`。）

**子条目圆点**:`•` 用 ink、或锈红小方块 `▪`。别用 emoji。

**渲染**:走 card-layout §8 管线;Chrome 命令追加 `--virtual-time-budget=12000` 确保 Google Fonts 加载完再截图,否则衬线会回退成系统字。

## 7. do / don't

- ✅ 标题够大、留白够狠、只有一个锈红。
- ✅ 英文内容优先用 Playfair(满分复刻);中文用思源宋 900。
- ✅ 一张最多一根手绘箭头。
- ❌ 不加第二强调色、不加纹理做旧、不用纯黑 #000、不居中堆满、不用系统宋体/黑体糊弄。
- ❌ 别把下半部填满 —— 空着才是这套的设计。

---

## 8. 适合 / 不适合

适合:步骤教程、清单体、方法论、知识干货、英语学习、成长、生活方式。
不适合:需要热闹多巴胺撞色的、重纹理做旧的、信息极密的长图。

## 9. 页面集(模块) · 标准 8 种

本套已配齐标准模块,每种一个 `page-*.html` + `sample-*.png`,连发时按内容挑用:

| 模块 | 文件 | 这套里的版式 |
|---|---|---|
| 封面 | `cover.html` | 品牌顶 + 大标题垂直居中 + 页脚 |
| 概念 | `page-concept.html` | 锈红「概念」标 + 大词 + 定义段 |
| 数据 | `page-data.html` | 超大锈红数字 + 单位 + 说明 |
| 对比 | `page-compare.html` | 标题 + ✕(灰)/✓(锈红)两行,细线分隔 |
| 步骤 | `inner-step.html` | 大数字编号 + 标题 + 要点 + 手绘箭头 |
| 金句 | `page-quote.html` | 大字金句(不加引号)+ 锈红短下划线 |
| 清单 | `page-list.html` | 标题 + 锈红编号逐条 + 细线 |
| 封底 | `page-back.html` | 大收尾句 + 锈红线 + 关注 CTA |

**通用**:所有页都 京华老宋一种字、单锈红、`brand 顶 / main 垂直居中(translateY -52%)/ foot 底` 三段;锈红只点睛(标签/数字/✓/线)。这是**其余 9 套也要照搬的页面集标准**(各自用自己的字体配色招牌实现这 8 种)。
