# 风格包 · 粗体大字海报 (bold-poster)

> 母题:**Bold Poster**——得意黑超粗压缩大字 + 暗底 + 单枚亮色。
> 灵魂:冲击、态度、宣言;一句话占满整张。

## 1. 配色
暗底 `--bg:#141414` · 文字 `--paper:#F2F0EA` · 单枚 `--accent:#F5D300`(亮黄)。
也可反过来:亮色满版底 + 黑字。总之**一个强色 + 黑白**。

## 2. 字体
标题/数字 **得意黑 Smiley Sans Oblique**(ZeoSeven 92,压缩斜体超粗);正文 思源黑体常规。
引入:`https://fontsapi.zeoseven.com/92/main/result.css` + `Noto+Sans+SC:wght@400;500`

## 3. 招牌动作
① **得意黑大字满版**(100-190px),关键词标黄;② 暗底单黄,极简;③ kicker 小字引子。

## 4. 两档
- 响亮档(封面/金句):得意黑巨字;
- **正文档**(长文):暗底 + **思源黑常规粗细**白字,黄点关键词(得意黑只留标题)。

## 5. 页面集(模块)
`poster.html`(封面)+ `page-body/concept/data/compare/steps/quote/list/back.html` + `page-img-*.html`。
- 数据=得意黑巨数字 + 黄单位;对比=两块明暗;步骤·清单=得意黑黄编号 + 细线。
- **图文**:暗底上图片小圆角(`radius:8px`),说明浅色;暗底让照片更跳。

## 6. do / don't
✅ 暗底单黄、得意黑满版、关键词标黄、正文转常规粗细。
❌ 不多色、标题别用细字、别塞长段(长段走正文档)。

## 7. 适合
自律/搞钱/清醒/态度/观点。不适合温柔治愈、安静文艺。
